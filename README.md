# YOLOv8 Object Detection (Image & Webcam)

This project demonstrates how to use a **pre-trained YOLOv8 model** to perform **object detection** on a single image or using a **live webcam feed**.  
The detected objects are highlighted with **bounding boxes, labels, and confidence scores**, and the final output image is saved as `results.jpg`.
